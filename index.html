Plan 1 Tahun ke Depan

Penerapan CSS Bootstrap dalam pengembangan portofolio ini merupakan tugas dari Pak Isa Hamdan, S.Kom selaku guru bidang lomba sekaligus direktur perusahaan. Pembimbing PKL: Cholilurrohman, S.Kom.

Beasiswa First Kampus Realistis Portofolio Prestasi
Fokus utama saya bukan sekadar diterima kuliah, tetapi memastikan saya bisa kuliah tanpa biaya atau dengan beban biaya seminimal mungkin.

Saat web ini dibuat, fokus utama saya adalah persiapan lomba, memperkuat portofolio prestasi, dan menyusun strategi kuliah tanpa biaya melalui beasiswa. Bisnis florist tetap menjadi bagian dari konteks rencana saya, tetapi fokus utama plan ini adalah lomba, pendidikan, dan pembiayaan kuliah. SNBP hanya peluang tambahan, bukan strategi utama.
